Analog Library Premium Edition™ v__VERSION__

It’s a simple way to view metadata about published papers. You give it a DOI and Analog Library Premium Edition™ gives you a fast-loading, unadorned HTML page describing that paper. Use a URL like this: 

http://__HOST__/doi/[YOUR FAVORITE DOI HERE]

For example, maybe you’d like to learn about the case for multicores?

Here are some ways to use Analog Library automatically:

  • Bookmarklet. Drag this bookmarklet analogify to your browser’s “favorites” bar for a one-click redirect.
  • Userscript. Install this userscript using a browser like Tampermonkey for zero-click redirects.
  • Kagi redirect. If you use Kagi as your search engine, add this rule to your redirects to rewrite result URLs to point to Analog Library: ^https://dl.acm.org/doi/(.*)|https://__HOST__/doi/$1

Analog Library Premium Edition™ is by Adrian Sampson. Check out the source if you want.
